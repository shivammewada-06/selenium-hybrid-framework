# Hybrid Selenium Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Apache POI
- Extent Reports

## Features
- Hybrid Framework (POM + Data Driven + Keyword Driven)
- Cross Browser Testing
- Screenshot on Failure
- Excel Data Driven Testing
- Extent Reporting

## How to Run
1. Clone repo
2. Open in IntelliJ
3. Run LoginTest.java
